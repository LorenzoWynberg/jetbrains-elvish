Elvish Module System

Modules in Elvish allow you to organize and share code between scripts. Any .elv file can be used as a module.

Using Modules

Import a module with the use statement:

  • use ./mymodule - Import from current directory
  • use str - Import built-in module
  • use github.com/user/repo - Import from URL

Exporting

All functions and variables defined at the module level are automatically exported and can be accessed with the module prefix:

use ./mymodule
mymodule:my-function arg1 arg2
echo $mymodule:my-variable

Learn more about Elvish modules at elv.sh/ref/language.html#modules
